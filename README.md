# Employee Leave Management System (Full Project)

This archive contains a full-stack sample project (backend + frontend) matching the Employee Leave Management System spec.

## Quick start

- Backend: set up `.env` from `backend/.env.example`, then run `npm install` and `npm run dev` in `backend/`.
- Frontend: set `REACT_APP_API_URL` in `frontend/.env.example`, then run `npm install` and `npm start` in `frontend/`.

Seed manager: run `node seed.js` in `backend` to create a sample manager (manager@example.com / password123).
